# claude-usage-monitor
A live terminal dashboard that tracks your Claude Code token usage in real time.
Reads session data directly from `~/.claude/projects/` JSONL logs — no API keys or configuration needed.

## Features

- **Token Usage** — Today and weekly totals with per-type breakdown (input, output, cache read, cache create)
- **Estimated Cost** — What your usage would cost at API pricing (Opus, Sonnet, Haiku)
